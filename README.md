# Chitter API Frontend Challenge

This is a prototype of the Chitter Frontend. 

The features include:
* Creating Users
* Viewing all Peeps 

Techology:
- React.js
- Tailwind
- Cypress
- Axios

## Get started

To start the development server:
```
npm install
npm run dev
```

To run the tests (E2E & Component test):
```
npx cypress open
```
